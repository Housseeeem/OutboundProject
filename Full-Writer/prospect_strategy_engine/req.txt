## 🛠️ Technologies utilisées
- **Framework :** LangGraph (StateGraph)
- **Extraction :** Hunter.io API
- **Recherche Web :** Tavily API
- **IA Locale :** Ollama (Modèle Mistral)
- **Interface :** Streamlit

4-Pour faire fonctionner l'intégralité du système, vous devez ouvrir 3 terminaux différents :

1️⃣ Terminal 1 : Le Serveur d'IA (Ollama)
Assurez-vous qu'Ollama est installé et lancez le modèle Mistral :

2️⃣ Terminal 2 : Le Backend (Le Graphe)
Ce terminal sert à vérifier les connexions ou lancer des tests isolés (Optionnel si vous utilisez l'App) :

3️⃣ Terminal 3 : L'Interface Utilisateur (Streamlit)
C'est ici que vous lancez l'application web pour l'utilisateur final :


streamlit run app.py
